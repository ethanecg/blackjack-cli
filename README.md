# 🃏 blackjack-cli
A program made to play a simple game of blackJack using the windows console. This program use a two deck and dont use shuffle mechanic like it is meant in real game, it pick a random crad from the whole deck without a specif
order. Also, this program do not allowed for double nor split. This project was made to practice my object oriented porgramming in lua. Finally, the goal was pretty much to familiarize more with the based lua so no librairies were used.
## 💻 Technologies
  - lua
## 🤑 features
  - fully functional game of black jack (no double nor split rule)
  - 2 deck
  - real life like reshuffing (reshuffle at 30%)

I started this project by making the classes first (player and game) then I made the main file. I try to remove as much repetition as possible by making alot of function in the game.lua.

## 🚥Running the project
To run the project in your local environnement, follow these steps:
  1. Clone the repository to your local machine.
  2. Download lua-5.5.0 or a higher version here : https://luabinaries.sourceforge.net/download.html 
  3. Extract the lua zip folder
  4. Open the extracted folder, take lua.exe and put it in your root folder (C:\)
  5. Open the command prompt and change your directory to where you download the repository (e.x., cd \Program Files\blackjack-cli)
  6. After change your directory in command prompt enter the command : lua main.lua
