# 🃏 blackjack-cli
A simple, lightweight Blackjack game played right in the Windows console.
This project was built from scratch with no external libraries to practice Object-Oriented Programming (OOP) in Lua and master the core language features.
## 💻 Technologies
  - lua
## 🤑 features
  - Full Game Loop: A fully functional game of Blackjack (standard rules apply, but without splitting or doubling down).
  - 2-Deck System: Simulates a standard shoe using two decks of cards.
  - Smart Reshuffling: Automatically reshuffles the deck once 30% of the cards are left, mimicking real-life dealer mechanics.
## 👨‍🍳 The process
I started this project by making the core classes first (Player and Game) then I made the main file. I try to remove as much repetition as possible by making alot of function in the Game class.
## 📗 What I learned
Through this project, I got comfortable with Lua's built-in functions and system logic. Coming from a C# background, it was a great exercise in translating OOP concepts from a compiled language to a scripting language, mapping out the differences, and expanding my programming toolkit for future Lua projects.
## 🚥 Running the project
To run the project in your local environnement, follow these steps:
  1. Clone the repository to your local machine
  2. Download lua-5.5.0 here : https://sourceforge.net/projects/luabinaries/files/5.5.0/
  3. Extract the lua zip folder
  4. Open the extracted folder, take lua.exe and put it in your root folder (C:\)
  5. Open the command prompt and change your directory to where you download the repository (e.x., cd \Program Files\blackjack-cli)
  6. After change your directory in command prompt enter the command : lua main.lua
